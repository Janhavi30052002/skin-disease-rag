# 🩺 Skin Disease RAG System

> An AI-powered skin disease diagnosis system that combines deep learning image classification with Retrieval-Augmented Generation (RAG) to provide explainable predictions and relevant medical knowledge.

> **⚠️ Project Status:** Work in Progress (WIP)

---

## 📌 Overview

This project aims to assist in the preliminary analysis of skin lesion images by:

- Classifying skin diseases using a trained deep learning model
- Retrieving medically relevant information using a RAG pipeline
- Generating explainable AI responses for predictions
- Displaying similar cases for better interpretability
- Providing an intuitive web dashboard for end users

This project is intended for educational and research purposes and is **not** a replacement for professional medical diagnosis.
